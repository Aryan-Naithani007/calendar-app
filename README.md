# 📅 Interactive Calendar Component

A modern, responsive calendar component built using React.
This project replicates a wall calendar UI with interactive features.

---

## 🚀 Features

* 📆 Dynamic Monthly Calendar
* 🔁 Month Navigation (Next / Previous)
* 🎯 Date Range Selection (Start → End)
* 📝 Notes for selected dates (stored in localStorage)
* 📌 Highlight Today's Date
* 🎨 Clean Dark UI
* 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* CSS3

---

src/

---

## ▶️ How to Run

```bash
npm install
npm start
```

Then open:
http://localhost:3000

---

## 💡 Key Functionalities

* Click once → Select start date
* Click again → Select end date
* Selected range is highlighted
* Notes are saved per date using localStorage
* Navigation buttons change month dynamically
